PROBLEM FIXED:

DEPLOY CRM:
1. Go to vercel.com/new
2. Import your repo
3. Configure:
   - Root Directory: frontend/crm
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
   - Install Command: npm install --legacy-peer-deps
4. Environment Variables:
   VITE_API_BASE_URL = https://api.pennyanddebt.in/api/v1/crm
5. Deploy

DEPLOY WEBSITE:
1. Go to vercel.com/new
2. Import your repo (same repo, new project)
3. Configure:
   - Root Directory: frontend/website
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
   - Install Command: npm install --legacy-peer-deps
4. Environment Variables:
   VITE_API_BASE_URL = https://api.pennyanddebt.in/api/v1/website
5. Deploy

IF STILL FAILS:
- Clear build cache in Vercel dashboard
- Check build logs for specific error
- Ensure Node version is 18.x in Vercel settings
- Try: npm install --force instead of --legacy-peer-deps

COMMON ERRORS:
"Framework not detected" → Ensure Root Directory is set
"npm install timeout" → Use --legacy-peer-deps flag
"Module not found" → Check import paths in code
"Build failed" → Check build logs for missing deps
