## üèõÔ∏è DEPARTMENT STRUCTURE

### 1. COUNSELLOR TEAM

**Department Code**: `COUNSEL`  
**Team Size**: 5-10  
**Reports To**: Operations Head

#### Purpose
First point of contact. Convert raw leads into qualified opportunities.

#### Core Responsibilities
- Answer incoming calls/WhatsApp messages
- Conduct empathy-driven initial assessment
- Collect emotional + financial snapshot
- Qualify leads based on eligibility criteria
- Assign qualified leads to Advisors
- Maintain lead database hygiene

#### Daily Activities
- 20-30 lead calls per day
- Lead qualification scoring
- CRM data entry
- Follow-up on unreachable leads
- Daily handover report to Advisors

#### Tools Used
- CRM (Lead Management Module)
- WhatsApp Business API
- Call Recording System
- Lead Scoring Dashboard

---

### 2. ADVISOR TEAM

**Department Code**: `ADVISOR`  
**Team Size**: 5-10  
**Reports To**: Operations Head

#### Purpose
Convert qualified leads into enrolled programs with complete financial planning.

#### Core Responsibilities
- Deep financial assessment
- Document collection (KYC, statements, loan docs)
- Loan inventory creation
- Income/expense analysis
- Debt-to-income ratio calculation
- Program design (SIP amount, duration, strategy)
- Customer education and enrollment
- Program agreement generation

#### Daily Activities
- 5-8 detailed case assessments
- Document verification
- Financial modeling
- Customer presentation calls
- Enrollment closures
- Handover to Operations

#### Tools Used
- CRM (Case Management Module)
- Financial Calculator
- Document Management System
- Program Simulator
- eSign Integration
